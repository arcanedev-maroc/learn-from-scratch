# Laravel Forum (Laracasts clone)

## To install

Edit the `.env` file (database and stuff).

Run these commands to install the project.

```bash
$ composer install
$ php artisan key:generate
$ php artisan migrate:install
$ npm install && bower install
$ gulp
```

And voila !
